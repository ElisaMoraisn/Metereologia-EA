# Metereologia-EA

Este projeto consiste em um protótipo de estação meteorológica conectada, que coleta dados de sensores de temperatura, umidade e pressão atmosférica usando um ESP32 e os envia para a nuvem (Firebase).

O repositório contém um site dinâmico, com:
- Visualização em tempo real dos dados da estação
- Gráficos interativos que mostram o histórico dos últimos registros
- Atualização automática, sem precisar atualizar a página

O objetivo é tornar o projeto palpável e acessível, permitindo que qualquer pessoa visualize os dados da estação meteorológica via navegador.
